SimpleLogicsFactory

object SimpleLogicsFactory : LogicsFactory(source)

An implementation of the LogicsFactory interface that creates Logics objects using an empty constructor of the appropriate type.

Used by default in LogicsProvider.

Usage:

val factory = SimpleLogicsFactory
val provider = LogicsProvider(factory = factory)
val logics = provider<FooLogics>.get("foo")
coroutineScope.launch {
    logics.state.collect {
        onState(it)
    }
}
