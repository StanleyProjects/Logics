create

abstract fun <T : Logics> create(type: Class<T>): T(source)

Creates a new instance of the given type.

Usage:

val factory = object : LogicsFactory {
    override fun <T : Logics> create(type: Class<T>): T {
        return type
            .getConstructor()
            .newInstance()
    }
}
